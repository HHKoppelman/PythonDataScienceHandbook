Subtracting group-wise mean
```python
df.groupby('key').transform(lambda x: x - x.mean())
```
Same:
```python
def center(x):
    x = x-x.mean()
    return x
df.groupby('key').transform(center)
```

Groupby & Aggregate
```python
df.groupby('key').aggregate({'data1': 'min',
                             'data2': 'max'})
```

Groupby and mapping
```python
df2 = df.set_index('key')
```
